# Patient Management System

A Python-based hospital patient management system with MySQL integration.

## Features
- Read patient data
- Show statistics
- Data visualization
- Add new patients

## Setup
1. Import `pasien_db.sql` to MySQL
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python src/project1.py`

## Database
MySQL database with patient records including:
- Patient_ID
- Full_Name
- Gender
- Age
- Diagnosis
